Instalacja
pip install -r requirements.txt
python symulacja.py

Zadaniem gracza jest jak najdłuższe utrzymanie przynajmniej trzech z czterech gatunków zwierząt (detrytusożercy, roślinożercy, mięsożercy, wszystkożercy)

w prawym górnym rogu znajduje się timer, zatrzymuje się gdy stracimy dwa gatunki

można zmieniać główne parametry jak liczba szybkość i zmysł. Przed odpaleniem programu tylko w kodzie,
po odpaleniu programu również suwakami

przycisk "od nowa" resetuje symulację i rozpoczyna kolejną, z tymi samymi parametrami co poprzednio
